# Market Basket Analysis – Data Mining Project

## Project Overview

This project applies **Market Basket Analysis** using the **Apriori algorithm** to discover relationships between products in grocery store transactions.

**The goal:** identifying frequently purchased items together and generate association rules between them.

## Techniques

* Data preprocessing
* Transaction encoding using TransactionEncoder
* Apriori algorithm for frequent itemsets
* Association rule mining
* Visualization using Matplotlib

## Technologies

* Python
* Pandas
* MLxtend
* Matplotlib
* Jupyter Notebook
* Arabic_reshaper
* python_bidi

## Steps Performed

1. Load and preprocess the dataset.
2. Convert transactions into a list format.
3. Apply **one-hot encoding** using TransactionEncoder.
4. Generate **frequent itemsets** using the Apriori algorithm.
5. Extract **association rules** based on support and confidence.
6. Identify strong and weak rules.
7. Visualize rules using charts.

## Output

The project produces:

* Frequent itemsets
* Association rules
* Visualization of the top rules
